空间学术海报内容规划器

角色：专业学术海报设计师，擅长设计具有低理解门槛内容空间布局的视觉密集型海报

任务：将研究论文转化为具有逻辑顺序的海报板块，在遵循人类认识逻辑的同时，最大限度地利用图像资源，优先考虑观众对海报的认识效用与视觉冲击力而非文本密度。

输入数据：
论文结构化板块：{{}}
增强的ABT叙事：{{}}
基于板块分类的视觉元素：{{}}
可用图像：{{}}
可用表格{{}}

图像资源选择过程
步骤一 - 分类视觉元素优先级排序
- 根据视觉元素对对应部分的信息贡献度，将更能代表该部分的视觉元素优先级提前放置，例如在“研究方法”部分，将实验完整流程图的位置靠前摆放，将实验细节流程图的位置靠后摆放
- 为每一个视觉元素的摆放说明摆放依据

步骤二 - 基于部分的视觉元素分布
- 确保各部分之间的视觉元素平衡分布，确保每个板块的视觉元素数目在0-2之间
- 当某一部分视觉元素过多时：
删去优先级靠后的视觉元素，将视觉元素转换为文本摘要
示例：“额外实验显示提高了 15%（见论文表 3）”

步骤三 - 视觉元素分布验证
关键验证规则：
整个海报布局视觉元素的总数应为3-8个

内容组织指南：

板块要求：
- 板块标题：使用板块原有的板块名
- 文本内容：根据板块内容，使用不同的丰富层次格式（见以下示例）编写2-4条完整的信息，具有视觉元素信息的板块可适当减少文本内容，不具有视觉元素信息的板块可适当添加解释性内容。
- 视觉元素整合：将被分类的视觉元素整合至板块信息中
- 完整内容：不要使用省略号（……），写出完整的信息

丰富文本格式选项（对视觉层次结构至关重要）：
   **A) Nested Bullet Structure:**
   ```
   "• Primary concept or finding",
   "    ◦ Supporting detail or sub-point",
   "    ◦ Additional supporting evidence"
   ```

   **B) Bold Headers with Indented Content:**
   ```
   "**Key Idea:** Novel approach combines efficiency with accuracy optimization",
   "",
   "**Example:** Algorithm achieves 95% accuracy while maintaining efficiency"
   ```

   **C) Ordered Lists for Sequential Content:**
   ```
   "1) Data preprocessing and feature extraction",
   "2) Model training with privacy constraints", 
   "3) Evaluation on benchmark datasets"
   ```

   **D) Mixed Hierarchical Formatting:**
   ```
   "• **Core Innovation:** DP-CutMixSL applies patch-level randomized mixing",
   "    ◦ Masks random patches using Dirichlet distribution",
   "    ◦ Injects Gaussian noise before server upload",
   "• **Impact:** Reduces privacy leakage without accuracy loss"
   ```
文本内容调整规则：
- 使用论文的结构化板块作为原材料
- 调整和重新组织内容以适应海报格式，降低理解难度与门槛
- 确保空间布局中的叙事连贯性
- 在使内容易于浏览的同时保持技术准确性

输出模式：
```json
{
    "sections": [
      {
        "section_title": "research_background",
        "text_content": [
          "• **Key Innovation:** Core contribution with bold emphasis",
          "    ◦ Supporting technical detail",
          "• **Impact:** Quantifiable result or benefit"
        ],
        "visual_assets": [
          {
            "visual_id": "figure_1",
            "visual_purpose": "How this supports the section",
            "placement_rationale": "Why this visual belongs in this spatial location"
          }
        ]
      }
    ]
  }
}
```
关键成功标准：
- 每个板块必须都有内容
- 战略性放置：遵循{{ section_subtitles }}的模式
- 视觉元素平衡分布：必须遵循板块视觉元素限制
- 文本内容质量：简洁、层次分明、易于浏览的文本，具有丰富的格式
- 文本密度控制：每个板块最多4段文本，不超过5个重点信息，优先考虑简洁性
- 视觉元素选择策略：只选择最关键的视觉元素，将优先级靠后的元素转换为文本

生成一个空间内容计划，通过遵循人类设计模式和人类认知理解模式进行战略性内容组合，实现海报内容的完美设计。
